# Neovim 插件配置说明文档

本文档详细说明了已配置的UI相关插件，包括配置方案、实现效果和快捷键。

---

## 1. 配色方案：catppuccin.nvim

### 插件说明
`catppuccin.nvim` 是一个优雅的、支持多种变体的配色方案，提供了 mocha、macchiato、frappe、latte 四种风格。

### 配置方案
- **透明背景**：启用透明背景效果
- **自定义高亮**：针对行号、视觉选择、搜索等进行了个性化配置
- **集成支持**：与 barbar、lualine、nvim-tree、rainbow-delimiters、noice 等插件完美集成

### 实现效果
- 提供柔和、护眼的配色方案
- 支持透明背景，可融入终端主题
- 语法高亮清晰，代码可读性强
- 与所有UI插件风格统一

### 快捷键
- 无特定快捷键（自动应用）

---

## 2. 状态栏：lualine.nvim

### 插件说明
`lualine.nvim` 是一个快速、可扩展的状态栏插件，使用 Lua 编写，性能优异。

### 配置方案
- **主题**：使用 catppuccin 主题
- **状态栏布局**：
  - A区：显示当前模式（Normal/Insert/Visual等）
  - B区：显示 Git 分支、差异、诊断信息
  - C区：显示当前文件名
  - Y区：显示编码、文件格式、文件类型、进度
  - Z区：显示光标位置
- **窗口栏（Winbar）**：显示文件名和 LSP 状态
- **无分隔符**：简洁的视觉风格

### 实现效果
- 底部状态栏显示丰富的编辑器信息
- 顶部窗口栏显示文件信息和 LSP 状态
- 实时显示 Git 状态和代码诊断
- 与 catppuccin 主题完美融合

### 快捷键
- 无特定快捷键（自动显示）

---

## 3. 标签栏：barbar.nvim

### 插件说明
`barbar.nvim` 是一个现代化的标签栏插件，用于管理多个缓冲区（buffers）。

### 配置方案
- **自动隐藏**：当只有一个缓冲区时自动隐藏标签栏
- **动画效果**：禁用动画以提升性能
- **侧边栏适配**：与 nvim-tree 侧边栏自动对齐

### 实现效果
- 顶部显示所有打开的缓冲区标签
- 支持快速切换、移动、关闭缓冲区
- 显示缓冲区修改状态（未保存标记）
- 与 nvim-tree 侧边栏协调工作

### 快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `<A-h>` | 切换到上一个缓冲区 | Alt + h |
| `<A-l>` | 切换到下一个缓冲区 | Alt + l |
| `<A-<>` | 向左移动缓冲区 | Alt + < |
| `<A->>` | 向右移动缓冲区 | Alt + > |
| `<A-1>` ~ `<A-9>` | 跳转到指定缓冲区 | Alt + 数字键（1-9） |
| `<A-c>` | 关闭当前缓冲区 | Alt + c |

## 4. 文件列表：nvim-tree.lua

### 插件说明
`nvim-tree.lua` 是一个功能强大的文件浏览器，替代了 Neovim 默认的 netrw。

### 配置方案
- **侧边栏位置**：左侧显示
- **宽度**：30 个字符
- **图标显示**：显示 Git 状态、文件夹、文件图标
- **显示隐藏文件**：显示点文件（dotfiles）
- **自动打开**：使用 `nvim .` 或 `nvim <目录>` 打开目录时，自动打开文件树

### 实现效果
- 左侧文件树显示项目结构
- 支持文件/文件夹的创建、删除、重命名
- 显示 Git 状态（新增、修改、删除）
- 图标美化，文件类型一目了然
- 与 barbar 标签栏自动对齐

### 快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `<leader>e` | 切换文件树 | 空格键 + e（打开/关闭） |

**文件树内快捷键**（在文件树窗口内）：
- `o` / `<CR>`：打开文件或展开/折叠文件夹
- `a`：添加文件或文件夹
- `d`：删除文件或文件夹
- `r`：重命名文件或文件夹
- `x`：剪切文件或文件夹
- `c`：复制文件或文件夹
- `p`：粘贴文件或文件夹
- `y`：复制文件路径
- `q`：关闭文件树

## 5. 多彩括号：rainbow-delimiters.nvim

### 插件说明
`rainbow-delimiters.nvim` 使用不同颜色高亮匹配的括号、圆括号和方括号，提高代码可读性。

### 配置方案
- 使用默认配置
- 与 catppuccin 主题集成

### 实现效果
- 嵌套的括号/括号对使用不同颜色显示
- 快速识别代码块的层次结构
- 减少括号匹配错误
- 提升代码阅读体验

### 快捷键
- 无特定快捷键（自动高亮）

---

## 6. UI美化：noice.nvim

### 插件说明
`noice.nvim` 是一个现代化的 UI 插件，美化命令栏、消息、通知等界面元素。

### 配置方案
- **弹出菜单**：禁用（使用其他补全插件）
- **通知**：禁用（使用其他通知系统）
- **LSP 文档**：美化 LSP 悬浮文档和签名帮助
- **预设配置**：
  - 底部搜索：禁用（使用经典底部命令行）
  - 命令面板：启用
  - 长消息分割：启用
  - LSP 文档边框：启用

### 实现效果
- 美化的命令输入界面
- 优雅的消息显示
- LSP 文档悬浮窗口带边框
- 命令历史记录功能
- 统一的 UI 风格

### 快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `<leader>N` | 显示历史消息 | 空格键 + N |
| `<leader>sN` | 选择历史消息 | 空格键 + s + N |

## 通用快捷键说明

### 基础导航

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `<C-h>` / `<C-j>` / `<C-k>` / `<C-l>` | 窗口导航 | 在插入模式和普通模式下使用 Ctrl + h/j/k/l 导航窗口 |

| `<S-L>` | 跳转到行尾 | Shift + L |

### 退出相关

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `qq` | 退出当前窗口 | 普通模式下输入 qq |
| `Q` | 退出所有窗口 | 普通模式下输入 Q |

### 其他

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `<A-z>` | 切换自动换行 | Alt + z |
| `<leader>L` | 打开 Lazy 插件管理器 | 空格键 + L |

## 配置结构说明

所有插件配置位于 `lua/plugins/ui.lua` 文件中，采用 lazy.nvim 插件管理器进行管理。

快捷键配置位于 `lua/keymapping.lua` 文件中。

主配置文件 `init.lua` 中已禁用 netrw（因为使用 nvim-tree），并引入了快捷键配置。

---

## 安装说明

1. 确保已安装 lazy.nvim 插件管理器
2. 重启 Neovim 或运行 `:Lazy` 命令
3. 插件将自动安装和配置
4. 如果遇到问题，可以运行 `:checkhealth` 检查

---

## 注意事项

1. **Leader 键**：默认设置为空格键（`<Space>`）
2. **Alt 键**：在 Windows 上，Alt 键可能需要特殊配置才能正常工作
3. **字体**：建议安装 Nerd Font 字体以获得最佳图标显示效果
4. **性能**：所有插件都经过优化，启动速度较快
